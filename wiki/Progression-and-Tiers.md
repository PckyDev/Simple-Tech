# Progression and Tiers

Simple Tech uses four major progression tiers.

## Tier 0 - Primitive Tech

Focus:

- manual tools
- low-throughput conversion
- starter materials

Key content:

- `primitive_hammer`
- grindstone raw-iron crushing
- anvil iron-plate pressing
- first tutorial objectives

## Tier 1 - Mechanical

Focus:

- first placeable machines
- fuel-powered processing
- first automation hooks

Key content:

- `crusher`
- `press`
- `mixer`
- `smelter`
- basic fuel use
- first machine loops

## Tier 2 - Industrial

Focus:

- multi-step production chains
- upgraded materials
- byproducts and recycling

Key content:

- `assembler`
- `refinery`
- `treated_plate`
- `reinforced_casing`
- `refined_flux`
- `control_core`

## Tier 3 - Advanced Automation

Focus:

- high-throughput chains
- routing and filters
- efficiency optimization
- longer advanced processing

Key content:

- `electrolyzer`
- `fabricator`
- `charged_mesh`
- `fabrication_matrix`
- `pristine_matrix`
- `precision_lens`
- `thermal_cell`

## Feature Set Mapping

### Feature Set A - Core Tech Loop

- items
- recipes
- main GUI
- crusher / press / mixer
- basic fuel

### Feature Set B - Industrial Expansion

- assembler
- refinery
- upgrades
- early logistics
- more advanced chains

### Feature Set C - Automation Depth

- priority routing
- throughput balancing
- ownership and access controls
- admin diagnostics

### Feature Set D - Polish & Stability

- balance pass
- documentation completion
- performance pass
- QA regression work

### Feature Set E - Onboarding UX Refresh

- quieter tutorial flow
- completion-first messaging
- `/st objective next`
- cleaner onboarding wording

## How Players Usually Advance

1. Primitive conversion
2. Basic machines
3. Fuel management
4. Item components and casings
5. Advanced machine registration
6. Routing and buffers
7. Tier 3 precision chains
