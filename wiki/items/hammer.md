# Hammer

## Summary
- **Tier:** Primitive Tech
- **Base item:** Mace
- **Purpose:** Manual processing tool for early progression

## Crafting Recipe
```text
[Iron Ingot] [Iron Ingot] [Iron Ingot]
[Iron Ingot] [Stick]      [Iron Ingot]
[Empty]      [Stick]      [Empty]
```

## Uses
- In an anvil with raw ore to make crushed ore.
- In an anvil with an ingot to make a plate.
- Not consumed during processing.
- Cannot be used as a weapon.

## Processing Recipes
- `Hammer + Raw Iron` → `2x Crushed Iron`
- `Hammer + Raw Gold` → `2x Crushed Gold`
- `Hammer + Raw Copper` → `2x Crushed Copper`
- `Hammer + Iron Ingot` → `Iron Plate`
- `Hammer + Gold Ingot` → `Gold Plate`
- `Hammer + Copper Ingot` → `Copper Plate`

## Notes
The Hammer is the entry point for the whole tech tree.
